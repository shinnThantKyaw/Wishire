## Phase Goal

**As a** birthday person, **I want to** read the wish one sentence at a time, see photos, and get a confetti celebration at the end, **so that** the experience feels like unwrapping a real gift.

## Acceptance Criteria

- [ ] Sentences reveal character by character on tap (typewriter effect)
- [ ] Skip button instantly shows full sentence
- [ ] Chime SFX plays on each sentence reveal
- [ ] After all sentences, photos appear as auto-advancing slideshow (3-4s)
- [ ] Slideshow has prev/next controls that pause auto-advance
- [ ] Confetti explosion + "Happy Birthday!" after last photo/sentence
- [ ] Replay button resets entire experience from gift box
- [ ] All AnimatePresence keys include playCount
- [ ] All variants at module level

     - GIFT_BOX: `<motion.div key={`gift-${playCount}`}><GiftBox .../></motion.div>`
     - REVEALING: `<motion.div key={`sentence-${state.sentenceIndex}-${playCount}`}><SentenceRevealer .../></motion.div>`

     - FINALE: `<motion.div key={`finale-${playCount}`}><ConfettiFinale .../></motion.div>`

   - SentenceRevealer receives: sentence from wish.sentences[state.sentenceIndex], onRevealed → dispatch NEXT_SENTENCE, onSkip → dispatch SKIP_TYPING
   - PhotoSlideshow receives: wish.photos, onComplete → dispatch START_FINALE
   - ConfettiFinale receives: wish.recipientName, onComplete → dispatch FINALE_DONE
   - ReplayButton receives: onReplay → dispatch REPLAY
