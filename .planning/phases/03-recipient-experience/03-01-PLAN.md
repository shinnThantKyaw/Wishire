## Phase Goal

**As a** birthday person, **I want to** open a link and see an animated gift box with my name on it that I can tap to start experiencing a personalized birthday wish, **so that** I feel the magic of receiving a real interactive gift.

## Acceptance Criteria

- [ ] /wish/:id route renders WishPage component
- [ ] WishPage fetches wish data from GET /api/wish/:id on mount
- [ ] Gift box displays sender name ("From [Name]") with theme-colored accents
- [ ] "Tap to open" hint visible below gift box
- [ ] Tapping gift box triggers split animation (~1s) with particle burst
- [ ] Background music starts on gift box tap (50-60% volume, loops)
- [ ] Pause/resume button in top-right corner toggles music
- [ ] SFX whoosh plays on box open
- [ ] State machine advances to UNWRAPPING after box animation completes
- [ ] Invalid wish ID shows error state with "Try again" button
- [ ] Network error shows error state with connection message
- [ ] Loading state shows skeleton shimmer while fetching
