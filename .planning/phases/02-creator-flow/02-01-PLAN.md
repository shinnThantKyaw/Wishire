## Phase Goal

**As a** wish creator, **I want to** drag-and-drop photos onto the form and select a theme by clicking color circles, **so that** I can customize my birthday wish with a smoother, more visual experience.

## Acceptance Criteria

- [ ] react-dropzone installed in client/package.json
- [ ] PhotoUploader renders a dashed-border dropzone with drag-active highlight
- [ ] PhotoUploader shows 80x80px thumbnail previews with remove buttons
- [ ] PhotoUploader caps at 5 files, shows "{count}/5" counter
- [ ] PhotoUploader revokes blob URLs on removal and unmount
- [ ] ThemeSelector renders 5 color circles (sunrise, ocean, lavender, forest, midnight)
- [ ] ThemeSelector shows theme name below selected circle
- [ ] All new CSS follows BEM-like naming and uses existing CSS custom properties
- [ ] New components are standalone (no CreatePage integration yet -- that is Plan 02)
